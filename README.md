# pixsoul@1.0.0


PixSoul - Login & Signup example


## Installation
Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install pixsoul --save
```

This package is provided in these module formats:

- CommonJS
